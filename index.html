🔑 Cómo configurar EmailJS (gratis, 200 emails/mes, sin backend)
1. Crea cuenta en emailjs.com
2. Crea un Email Service (Gmail, Outlook, etc.) → copia el Service ID
3. Crea un Email Template con variables {{to_email}}, {{subject}}, {{message}}, {{to_name}}
4. Ve a Account > API Keys → copia tu Public Key
