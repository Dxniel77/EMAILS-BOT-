Solución: Email llega vacío en Gmail

Este es el problema más común con EmailJS. El template debe usar {{{message}}} con 3 llaves (no 2) y tener Content type: HTML activado.

  • Ve a emailjs.com → Email Templates → edita tu template
  • En el campo Body / Content escribe: {{{message}}}
  • En opciones del template activa "Content type: HTML"
  • Haz clic en Save y vuelve a enviar
🔑 Cómo obtener las credenciales (gratis · 200 emails/mes)
1. Crea cuenta en emailjs.com
2. Email Services → conecta tu Gmail → copia el Service ID
3. Email Templates → crea template → copia Template ID
4. Account → API Keys → copia tu Public Key

📋 Template correcto para EmailJS

Crea un template en emailjs.com con exactamente este contenido:

To:       {{to_email}}
Subject:  {{subject}}
Body:     {{{message}}}

⚠️ IMPORTANTE: Usa {{{message}}} con 3 llaves para HTML
⚠️ Activa "Content type: HTML" en las opciones del template
✅
¿Por qué 3 llaves?
Con 2 llaves {{message}} EmailJS escapa el HTML y lo envía como texto plano (lo que ves en pantalla). Con 3 llaves {{{message}}} lo envía como HTML real con el diseño correcto.
❓ Guía de uso rápido

1. Importar contactos
Ve a Contactos y arrastra el Excel exportado de EOS Community. El bot detecta automáticamente los campos.

2. Gestionar inactivos
Ve a Inactivos para ver todos los usuarios con estatus Inactivo. Habilita los que quieres incluir en campañas. Cuando respondan, márcalos como "Suscrito".

3. Crear una campaña
Ve a Campaña, selecciona a quién enviar, elige una plantilla visual o pega HTML propio, personaliza el contenido y envía.

4. Variables disponibles
{{nombre}} {{email}} {{ciudad}} {{empresa}} {{pais}}

5. GitHub / Publicar online
Sube este archivo a GitHub y activa GitHub Pages para tener una URL permanente. O arrastra el archivo a Netlify Drop para una URL instantánea.
